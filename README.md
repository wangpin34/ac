# AC
AC is a tiny json viewer which can be pluged in your pages,web viewers

## Install
```
bower install ac
```
## Usage

Just put valid json string into html tab like following
```
<div class="json-viewer">
{"name":"wangpin","home":"weinan","children":["Emma","Bob"]}
</div>
```
